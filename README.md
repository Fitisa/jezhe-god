Этот репозиторий содержит таблицу с примерами конструкции типа _еже+год_ в среднерусских текстах.

Таблица подготовлена в качестве электронного приложения к статье:

_Фитискина А. А._ _Еже год/годов/годь_: загадки одной среднерусской конструкции // Русский язык в научном освещении. 2026. (в печати)

Таблица включает все примеры, обсуждаемые и подсчитываемые в работе.

Содержание

jezhegod.csv — таблица со всеми собранными примерами

Она включает:
- cам пример,
- источник,
- точную и условную дату,
- минимальную грамматическую разметку (существительное, использованное в конструкции, падеж, число, часть речи),
- информацию о том, является ли пример клишированным.

---

This repository contains a table with examples of the _ježe+god_ construction in Middle Russian texts.
The dataset was prepared as part of the article:

Fitiskina, Anna A. (2026). The mysteries of a Middle-Russian construction ježe god/godov/god’. _Russian Language and Linguistic Theory_ (in print)

The table includes all examples discussed and counted in the paper. 

Contents

jezhegod.csv — a table containing all collected examples.

It includes:
- the example itself,
- the source,
- precise and approximate dating,
- minimal grammatical annotation (the noun used in the construction, case, number, part of speech),
- information on whether the example is formulaic.
